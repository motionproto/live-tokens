How the geometry works.

The bar lives in a position: absolute slot inside a wrapper that has padding-left equal to gap + bar-width. In its linked rest position the bar sits at left: 0 + gap — flush with the control's left edge. In its unlinked position it translates by -(gap + bar-width), which lands it exactly one gap-width to the left of the control. The control itself doesn't move; only the bar slides. That's what gives the "pops off" feel.

The transition uses a slight overshoot easing (cubic-bezier(.5, 1.6, .5, 1)) so the bar lands with a tiny bounce — the literal "bloop." Color crossfade from teal to yellow is on a softer curve so it doesn't read as flicker.
