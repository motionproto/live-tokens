# Archived documentation

These are the original developer-reference chapters (architecture, state,
tokens and themes internals, the component system, the dev-server plugin, the
overlay and router, recipes, conventions) plus the token-naming appendix.

They are **not rendered on the `/docs` site**, which is now a short
user-facing guide shipped with the package (see `src/editor/docs/content/`).
They are kept here for maintainers and may be out of date. Treat the package
source as the source of truth.
