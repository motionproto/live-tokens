# @motion-proto/create-live-tokens

Scaffold a new [`@motion-proto/live-tokens`](https://www.npmjs.com/package/@motion-proto/live-tokens) app.

```bash
npm create @motion-proto/live-tokens@latest my-app
cd my-app
npm install
npm run dev
```

This is a thin initializer: all scaffolding (the template and version-matched
token seeds) lives in `@motion-proto/live-tokens` and is reused via its
`/create` export. The same scaffold is also available without this package as
`npx @motion-proto/live-tokens create my-app`.
